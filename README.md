# Task Manager API 🚀

API CRUD de tarefas em TypeScript com Docker.

## ✨ Funcionalidades
- Criar, listar, atualizar e deletar tarefas
- Containerizada com Docker
- Pronta para produção

## 🛠️ Tecnologias
- TypeScript
- Express.js
- Docker
- Postman (para testes)

## 🚀 Como Executar

### Localmente
```bash
npm install
npm run dev

docker-compose up --build (Com Docker)
